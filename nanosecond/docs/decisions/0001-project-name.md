# ADR 0001: Project is named "Nanosecond"
Date: 2026-07-03 · Status: accepted

## Context
Working title "Grounded" collides with Obsidian's 2020 game. Shortlist: Load Bearing,
Headroom, Back of the Envelope, Nanosecond.

## Decision
**Nanosecond** — after Grace Hopper's 30cm wires (the distance light travels in 1ns),
which is also the game's opening toy (Race Light).

## Consequences
All player-facing branding, package name, and repo naming use Nanosecond. The Hopper
story leads the README and should eventually appear in-game.
