# 900 — Backlog (v2 — DO NOT BUILD)
Design intent only, in priority order (reordered by ADR 0005 — migration under
live traffic is the staff-level skill no interview-prep resource teaches; the
meter measures, Legacy teaches):

1. **Legacy campaign** — persistent company; each chapter's growth breaks the
   previous chapter's architecture; refactor under LIVE traffic. The curriculum
   is the migration playbook itself: expand/contract schema changes, dual-write,
   shadow reads, cutover with rollback — every step while the request counter
   keeps ticking and the error budget is real.
2. **Readiness Meter** — rubric-mapped mock exam producing a per-dimension
   readiness report.

Promote to a numbered spec only after 000–140 are done and playtested.
